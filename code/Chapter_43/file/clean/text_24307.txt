A quick word about selfpromotion

Please do not add advertising or inappropriate external links to Wikipedia Wikipedia is not a mere directory of links nor should it be used for advertising or promotion  Inappropriate links include but are not limited to links to personal web sites links to web sites with which you are affiliated and links that exist to attract visitors to a web site or promote a product  See the external links guideline and spam policies for further explanations of links that are considered appropriate  If you feel the link should be added to the article then please discuss it on the  rather than readding it See the welcome page to learn more about Wikipedia Thank you
